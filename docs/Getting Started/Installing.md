# Requirements
- [NodeJS 18.0.0](https://nodejs.org/en/download/) or higher

# Installation

Create a project or directory for your bot, and change to that directory in your command line. Then use NPM to install Yags

```
npm install yags.js
```

Yags is now installed on your system.
