## What is Yags?

A lightweight, fast, and minimal Node.js library for interacting with the Discord API.

Yags is a fork of Eris, optimized for performance and simplicity by removing voice features and streamlining the codebase.

## Features

- Lightweight and minimal footprint
- Fast and efficient Discord API interactions
- Full support for Discord's REST API and Gateway
- Built-in command framework
- Sharding support
- & more!
